Calculator Project (Normal & Optimized Version)
This mini project helped me clear the concepts I covered from the day I started my internship. In this project, our industrial mentor gave us requirements to build a calculator. I know it's a small project, but it gave me more confidence to move forward.

What Input Does It Take?

It takes 2 number inputs from the user to perform calculations.
It takes 1 string input from the user as an operator to decide which operation to perform.
It asks the user whether they want to continue or not.


What Output Does It Give?

It gives the Result of the calculation.
It gives "Invalid Number" if any number is zero in Division.
